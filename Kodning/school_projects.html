Entrepreneurial project

In this project we had to develop a website with our own product. Our product was that we wanted to create a website that could help start-up companies get started with a package in which you could get a logo, a website, clothes and a teaser video. We also had another package where you could choose the parts you were missing for your company, so you did not have to pay for the whole package. My role in this project was that I was the team leader and had to be the one to insure agreements and insure that our plan was being followed. In this project I was the one designing the logo as well as the payment page.

To make this project the following programs was used:

  • HTML
  • CSS
  • Adobe illustratior
  • Adobe XD
